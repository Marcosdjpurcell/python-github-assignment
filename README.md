# Study Time Tracker

## Description

Study Time Tracker is a simple Python application that helps students estimate their weekly study hours by entering the amount of time studied in a day. It provides instant feedback and helps you plan your week more effectively.

## How to Run

Make sure you have Python installed. In your terminal:

```bash
python app.py
# or
python3 app.py
```

## Example Output

```
Welcome to my Python program!
How many hours did you study today? 2
You are on track to study 14.0 hours this week.
```
